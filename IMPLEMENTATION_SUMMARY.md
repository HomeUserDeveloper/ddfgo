## 🔄 ЖИЗНЕННЫЙ ЦИКЛ ВЕРСИИ

### Процесс компиляции

```
1. build.bat запускается
   ↓
2. Читает version.txt: 000.000.000.001
   ↓
3. Увеличивает Build: 000.000.000.002
   ↓
4. Сохраняет в version.txt: 000.000.000.002
   ↓
5. Компилирует с ldflags:
   go build -ldflags "-X main.Version=000.000.000.002" -o ddfgo.exe
   ↓
6. Переменная main.Version ЗАМЕНЯЕТСЯ на "000.000.000.002"
   ↓
7. ddfgo.exe создан с встроенной версией 000.000.000.002
   ↓
8. ddfgo.exe -version выводит: "ddfgo версия: 000.000.000.002"
```

### Каждый следующий раз

```
build.bat
   ↓
version.txt: 000.000.000.002 → 000.000.000.003
   ↓
go build с -X main.Version=000.000.000.003
   ↓
ddfgo.exe обновлен с новой версией
```
