### 2️⃣ Версионирование встроено в код (поддержано)

```bash
# Статическая переменная
var Version = "000.000.000.001"

# Встраивается при компиляции
go build -ldflags "-X main.Version=000.000.000.002" -o ddfgo.exe

# Вывод версии
ddfgo.exe -version  →  "ddfgo версия: 000.000.000.002"
ddfgo.exe -v        →  "ddfgo версия: 000.000.000.002"
```
