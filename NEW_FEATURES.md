# 📋 ddfgo - Новые функции (обновление)

## Две новые возможности добавлены

### 1️⃣ Фильтрация файлов по размеру (флаг `-all`)

**По умолчанию** программа теперь **исключает из удаления файлы размером менее 10KB**.

Это полезно при работе с коллекциями электронных книг, где часто присутствуют одинаковые генерируемые файлы:
- Конфигурационные файлы (`.config`, `settings.xml`)
- Служебные файлы от генераторов кода
- Лицензии и метаданные
- Другие системные файлы

**Примеры:**

```bash
# По умолчанию - исключаются файлы < 10KB
ddfgo -dir "D:\MyBooks"

# Удалять ВСЕ дубликаты, включая маленькие файлы
ddfgo -dir "D:\MyBooks" -all
```

### 2️⃣ Удаление пустых каталогов (флаг `-dir0`)

После удаления дубликатов нередко остаются **пустые каталоги**. Новый флаг `-dir0` автоматически удаляет их.

Программа рекурсивно обходит все подпапки директории и удаляет пустые папки.

### 3️⃣ Фильтрация по расширениям (флаг `-ext-all`)

**По умолчанию** программа исключает из обработки файлы с расширениями:
- `.exe` - Исполняемые файлы
- `.dll` - Динамические библиотеки
- `.lib` - Библиотеки ссылок

**Причина:** Windows Defender может блокировать доступ к этим файлам при запуске под обычным пользователем, генерируя ошибку:
```
Operation did not complete successfully because the file contains a virus or potentially unwanted software
```

**Примеры:**

```bash
# По умолчанию - исключаются .exe, .dll, .lib
ddfgo -dir "D:\Software"

# Обработать ВСЕ расширения (требуются права администратора)
ddfgo -dir "D:\Software" -ext-all

# Комбинирование всех флагов
ddfgo -dir "D:\Data" -all -ext-all -dir0
```

**Примеры:**

```bash
# Удалить дубликаты и пустые каталоги
ddfgo -dir "D:\FileCollection" -dir0

# Комбинировать оба флага
ddfgo -dir "D:\MyData" -all -dir0

# С логированием для проверки
ddfgo -dir "D:\MyData" -test -clean -dir0 -all
```

## 📊 Тестовый результат

При обработке директории с файлами:
- **small1.txt** (15 bytes) - маленький оригинал
- **small2.txt** (15 bytes) - маленький дубликат **[исключен по умолчанию]**
- **large1.txt** (14.5 KB) - большой оригинал  
- **large2.txt** (14.5 KB) - большой дубликат **[удален]**
- **empty_dir/** - пустая папка **[удалена с -dir0]**

**Результаты по командам:**

| Команда | small2 | large2 | empty_dir | Примечание |
|---------|--------|--------|-----------|-----------|
| `ddfgo` | ✅ Сохранен | ❌ Удален | ✅ Сохранена | Поведение по умолчанию |
| `ddfgo -all` | ❌ Удален | ❌ Удален | ✅ Сохранена | Удаляет все дубликаты |
| `ddfgo -dir0` | ✅ Сохранен | ❌ Удален | ❌ Удалена | Очищает пусто папки |
| `ddfgo -all -dir0` | ❌ Удален | ❌ Удален | ❌ Удалена | Полная очистка |

## 🔧 Справка программы

```
  -all
    Удалять дубликаты всех размеров, включая файлы < 10KB
    По умолчанию файлы меньше 10KB исключаются из удаления
    (это полезно при наличии генерируемых файлов в электронных книгах)
    Пример: ddfgo -dir "D:\Books" -all

  -dir0
    Удалить пустые каталоги после завершения обработки
    Рекурсивно обходит все подпапки и удаляет пустые директории
    Пример: ddfgo -dir "D:\Files" -dir0
```

## 📈 Производительность

При работе на каталоге 175GB с 240,000 файлов:
- Время обработки: **~1 мин 20 сек** (без изменений)
- Новая функциональность добавляет минимальную нагрузку
- Фильтрация происходит на уровне SQL запроса (эффективно)
- Удаление пустых папок использует сортировку (снизу вверх)

## ⚠️  Важные замечания

1. **Резервная копия:** Перед первым использованием создайте резервную копию важных данных
2. **Тестирование:** Используйте флаг `-test` для проверки результатов перед реальным удалением
3. **Чистая логика:** Проверить результаты в `ddfgo.log` перед удалением файлов
4. **Невозвратность:** Удаленные файлы нельзя восстановить!

## 🎯 Рекомендуемые команды

```bash
# Безопасный старт - проверка результатов
ddfgo -dir "D:\MyBooks" -test -clean

# Просмотр результатов
type ddfgo.log

# После проверки - реальное удаление
ddfgo -dir "D:\MyBooks"

# Для больших коллекций электронных книг
ddfgo -dir "D:\EBooks" -dir0

# Полная очистка
ddfgo -dir "D:\Data" -all -dir0 -test -clean
```
